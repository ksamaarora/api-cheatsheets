# API Cheatsheets

## Folder map
- `01-intro-to-apis` — what APIs are, common use cases, JSON vs XML basics
- `02-http-and-terminology` — HTTP verbs, status codes, headers, idempotency, caching semantics
- `03-api-styles` — REST, GraphQL, gRPC, Webhooks, SOAP
- `04-authentication` — API keys, OAuth 2.0, JWTs, HMAC, mTLS
- `05-documentation-tools` — OpenAPI/Swagger, Postman collections, mocking tools
- `06-api-features` — pagination, filtering, sorting, versioning, rate limits
- `07-performance-techniques` — caching strategies, batching, compression, CDN notes
- `08-api-gateways` — routing, throttling, auth offload, observability
- `09-frameworks` — common server/client frameworks in your stack (e.g., Express/FastAPI/Spring/Go)
- `10-integration-patterns` — webhooks vs polling, retries, idempotency keys, circuit breakers
- `cheat-sheets` — single-page summaries you can print or keep open
- `diagrams` — sequence diagrams, flow charts, architecture sketches
- `sample-projects` — tiny working examples to test ideas
